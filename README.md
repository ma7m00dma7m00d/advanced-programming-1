## Run Locally

Clone the project

```bash
  git clone https://github.com/ma7m00dma7m00d/advanced-programming-1
```

Go to the project directory

```bash
  cd advanced-programming-1
```

Select lecture

```bash
  cd src
  cd Lecture[01-02-03-04-05]
```

Install dependencies

```bash
dotnet restore
```

Run

```bash
dotnet run
```
